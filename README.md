# Todo API

Kleine REST-API zur Verwaltung von Todos, entwickelt mit Spring Boot.

## Technologien
- Spring Boot
- PostgreSQL
- Spring Data JPA
- Testcontainers
- JUnit
- Lombok

## Funktionen
- CRUD-Operationen für Todos
- Persistenz mit PostgreSQL
- Integrationstests mit Testcontainers

## Geplante Erweiterungen
- Containerisierung und Deployment mit Kubernetes
- Angular-Frontend
